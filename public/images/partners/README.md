# Logos dos Parceiros

Esta pasta contém os logos dos parceiros da Ku Kula Devz.

## Arquivos necessários:

1. **umum-logo.png** - Logo da Universidade Metodista Unida de Moçambique
2. **movimento-cidadania-logo.png** - Logo do Movimento pela Cidadania

## Especificações recomendadas:

- **Formato**: PNG com fundo transparente
- **Tamanho**: Aproximadamente 200x100px (proporção 2:1)
- **Qualidade**: Alta resolução para boa visualização
- **Fundo**: Transparente ou branco

## Como adicionar:

1. Salve os logos nesta pasta com os nomes exatos mencionados acima
2. Os logos aparecerão automaticamente na seção "Parceiros" do site
