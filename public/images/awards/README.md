# Fotos dos Prêmios da Ku Kula Devz

Esta pasta contém as fotos dos prêmios e reconhecimentos recebidos pela equipe.

## Arquivos necessários:

1. **premio-1.jpg** - Primeira foto do prêmio (formato vertical/retrato)
2. **premio-2.jpg** - Segunda foto do prêmio (formato quadrado/paisagem)

## Especificações recomendadas:

- **Formato**: JPG ou PNG
- **Tamanho**: 
  - premio-1.jpg: Aproximadamente 400x500px (formato retrato)
  - premio-2.jpg: Aproximadamente 400x400px (formato quadrado)
- **Qualidade**: Alta resolução para boa visualização
- **Conteúdo**: Fotos da equipe recebendo prêmios, certificados, troféus, cerimônias

## Como usar:

1. Adicione as fotos nesta pasta com os nomes exatos: `premio-1.jpg` e `premio-2.jpg`
2. As fotos aparecerão automaticamente na seção "Quem Somos" do site
3. Se as fotos não existirem, o sistema usa imagens placeholder automaticamente

## Contexto:

Essas fotos aparecem junto com a estatística "+150 mil Meticais em Prêmios" para mostrar os reconhecimentos que a Ku Kula Devz recebeu.
