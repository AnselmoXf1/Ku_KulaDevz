# Logos dos Patrocinadores

Esta pasta contém os logos dos patrocinadores da Ku Kula Devz.

## Arquivos necessários:

1. **vodacom-logo.png** - Logo da Vodacom Moçambique
2. **bci-logo.png** - Logo do BCI (Banco Comercial e de Investimentos)
3. **moztech-logo.png** - Logo da MozTech Solutions

## Especificações recomendadas:

- **Formato**: PNG com fundo transparente
- **Tamanho**: Aproximadamente 200x100px (proporção 2:1)
- **Qualidade**: Alta resolução para boa visualização
- **Fundo**: Transparente ou branco

## Como adicionar:

1. Salve os logos nesta pasta com os nomes exatos mencionados acima
2. Os logos aparecerão automaticamente na seção "Patrocinadores" do site

## Nota:

Se você quiser usar patrocinadores diferentes, edite o arquivo `constants.ts` na raiz do projeto e atualize a lista de parceiros com `type: "sponsor"`.
